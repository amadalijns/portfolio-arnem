Frank de Pratende Bank

Voor ons vak Project 4.0 kregen we de opdracht om voor Rurant een pratende AI-bank te ontwikkelen. Ons team bestond uit zes leden, verdeeld over drie IT-richtingen: drie applicatieontwikkelaars, twee AI-studenten en ik als cloud- en securitystudent.

Even wat achtergrondinformatie: Frank de Pratende Bank is een unieke, laagdrempelige manier om inwoners te betrekken bij de toekomst van hun publieke ruimte. Met behulp van AI kan deze slimme bank gesprekken voeren en vragen stellen over hoe mensen hun dorpsplein, pastorijtuin of park in de toekomst willen zien.

Technische aspecten

Om Frank de Pratende Bank optimaal te integreren, ontwikkelden we een oplossing met een Raspberry Pi en een USB-telefoonhoorn. De Raspberry Pi verwerkt spraak met speech-to-text (STT) en text-to-speech (TTS) software en stuurt de tekst via een API naar een cloud-AI-dienst, die de antwoorden van Frank genereert. De telefoonhoorn biedt meer privacy en voorkomt verstoring door omgevingsgeluiden zoals wind.

Onze applicatieontwikkelaars bouwden een gebruiksvriendelijk dashboard om vragen, antwoorden van bezoekers en instellingen van de bank te beheren. Ze ontwikkelden ook een API waarmee de Raspberry Pi gegevens uitwisselt en kan communiceren met de rest van het project.

De opstelling is volledig plug-and-play: zodra de Raspberry Pi wordt aangesloten, start hij automatisch op en start alle software. LED-indicatoren tonen de status van Frank: opstarten, gereed of luisteren.

Rolverdeling

Mijn rol in dit project was het hosten van de applicaties van onze ontwikkelaars (dashboard, API en database) in de cloud. Daarnaast heb ik een geautomatiseerd proces opgezet om van de verschillende soorten code Docker-images te maken, deze te testen en bij een succesvolle test automatisch te deployen in onze cloudomgeving via CI/CD-pipelines.
